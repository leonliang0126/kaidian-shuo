# 《开店说》借款结果 & 危机应对弹窗叙事文案（修订稿 v0.2 · 待老板审阅）

> 纯文案草稿，供老板审阅调性，未改动任何源文件、未写代码。
> 档位口径：亲友拒绝率按"成功借款次数"升级（0次=30% / 1次=70% / 2次+=95%），被拒不升级。

**v0.2 相对 v0.1 的调整**：把每条从"说明式旁白"改成"有声音/物件/小动作的戏"——信贷员加"背台词又多嘴"的官僚吐槽腔，发小加"一边骂你一边转账"的反转，高利贷加"顺手喝你奶茶 / 烟按进纸杯"的细思极恐钩子，亲友被拒塞进"一句让人心里一咯噔"的扎心台词；多用哒声、油渍、冰柜嗡声等小细节让场景立得住。

## 1. 银行（bank）
*设定：周息 4%，确定性填平缺口（缺口+1万缓冲），偏正规；前 2 次危机可借，第 3 次起锁定。语气：官僚吐槽体，信贷员像背台词又忍不住多嘴，冷面里藏点人情。*

**借到**
- **银行·借到-A**：信贷员把回单推过来，指尖在"已批"上哒哒敲了两下：「额度顶到缺口了，按月还——别让我月底替你挨批。」背完台词，他多嘴了这一句。
- **银行·借到-B**：柜员盖章时手一抖，章斜了。她没重盖，只把单子转过来：「批了。歪就歪吧，反正钱不歪。」你盯着那枚歪章，竟有点想笑。
- **银行·借到-C**：审批短信响时你正盯着红字发愁。隔着玻璃，柜员用口型比了句"下次未必有这窗口"，低头继续嗑瓜子。

**被拒（罕见：风控/评分不足）**
- **银行·拒-风控-A**：「系统退件：综合评分不足。」信贷员把屏幕转给你，耸肩：「这回连我都想替你求情——可它不认人。」

## 2. 亲友（private）
*设定：周息 12%，随机 5万–15万；拒绝率按"成功借款次数"递增（成功 0 次=30% / 1 次=70% / 2 次+=95%）。语气：又暖又扎心，拒绝要有具体难处 + 一句让人心里一咯噔的话。*

**借到**
- **亲友·借到-A**：发小一边在语音里骂你「又作死」，一边把转账截图甩过来：「十万，拿去。你要是再亏，我真不认你这兄弟了。」骂完，钱到了。
- **亲友·借到-B**：表姐二话没说转了账。你正要谢，她语音追过来：「别跟你姐客气。对了——我闺女下月学费你先别惦记啊，上次那笔妈还念着呢。」暖意一下凉了半截。
- **亲友·借到-C**：老同学沉默三秒，回了个"周转 OK"，又补：「别声张，我那口子正查我私房钱。」钱到了，像做贼。

**被拒（成功 0 次档 · 30%）**——态度仍亲近，留"下次也许能成"的余地
- **亲友·拒-30%-A**：表姐在电话那头叹气：「不是不帮，小叔上月手术刚垫了八万……等下月奖金姐再想招。」顿了顿又补：「你先撑住啊，姐在呢。」
- **亲友·拒-30%-B**：发小挠头：「真不巧，闲钱刚锁进理财半年。你先问别人，我赎回一笔立马找你。」他真把赎回申请截图发你了。
- **亲友·拒-30%-C**：老同学回得飞快：「这周腾不出手，我妈装修压着我钱呢。你再撑撑，下回优先你。」末尾多了个抱拳表情。

**被拒（成功 1 次档 · 70%）**——明显更疏远/为难，关系开始紧绷
- **亲友·拒-70%-A**：发小沉默好久：「上次那笔你还没动静呢……这回真帮不了，你别光盯我一个啊。」他没挂，却也没再说话。
- **亲友·拒-70%-B**：表姐语气凉了半截：「你这都第几回了？我家也不是开银行的。你上次说月底还的——」月底早过了。
- **亲友·拒-70%-C**：老同学发了个"难"字，又补：「兄弟，不是不仗义，我媳妇已经问我三回了，昨晚差点掀桌。」

**被拒（成功 2 次+ 档 · 95%）**——几乎彻底拒绝，关系裂痕、带怨气
- **亲友·拒-95%-A**：发小直接摁了语音：「别再找我了。你自己挖的坑自己填——我家钱也不是大风刮来的。」说完，把你移出了家庭群。
- **亲友·拒-95%-B**：表姐回得干脆：「没有。真没有。你再开这个口，年都不用来了。」最后一句，比钱还冷。
- **亲友·拒-95%-C**：老同学最后一条：「我算看明白了，你拿我当提款机。以后这种事，别开口了。」头像灰了。

## 3. 高利贷（predatory / 周转）
*设定：利率逐笔飙升（36%→54%→81%…，UI"周转 X%"），随机 5千–8万。语气：江湖黑话 + 细思极恐，放款人每句都像在埋钩子，笑着吓人。*

**借到**
- **高利贷·借到-A**：穿皮夹克的中年人把信封拍桌上，顺手把你那杯奶茶吸溜一口：「利息嘛，下周再说——哥不会让你一下子死透。」你盯着空杯，喉咙发紧。
- **高利贷·借到-B**：「手机响了就接，晚一天，数字就不一样咯。」对方挂得比你还快。你盯着到账的五千，听见自己咽口水的声儿。
- **高利贷·借到-C**：放款哥们儿递烟你没接，他把烟按进一次性纸杯，吐个烟圈：「这把三十六，下把五十四，再下把……你最好别走到那步。」钱到了，钩子也下了。

**被拒（可选极端情形：对方判定你已无油水可榨）**
- **高利贷·拒-无油水-A**：「你这摊子，榨也榨不出二两油了。」对方把烟按灭，「不借。借给你，哥怕连本都收不回。」
- **高利贷·拒-无油水-B**：电话那头笑一声：「不是不帮，是你连利息零头都还不起——这种单子，哥不接，接了晦气。」

## 4.（可选加分项）危机应对行动 flavor
*危机面板里"非借款"行动的弹窗短故事，是否一并采用由老板决定。各条对应既有数值效果（见 crisisActionDefs.ts）。*

**变卖设备（sell_equipment，+¥5,000）**
- **可选·卖设备-A**：二手贩子绕着机器东摸西看，指节敲得外壳咚咚响：「这磨损、这锈、这年份不行——电机都响了吧？」你心里那八千的预期，被他一路砍到五千。急用钱套现续命，你只能点头。他数钱的手都没停，扛机器出门还补一句：「早点卖对了，再放俩月只值这半价。」
- **可选·卖设备-B**：你本以为这台能卖个好价。对方一搭眼你那神色就笑了：「急出手吧？对半。」你牙咬碎了也只能点头，笔尖在转让单上顿了顿。机器被拖走，你心里骂自己一句：早干嘛去了。

**清仓促销（clearance_sale，+¥2,000）**
- **可选·清仓-A**：你把临期品码成"全场五折"，扫码声噼里啪啦。有人凑近嘀咕「原来平时赚这么狠」，你笑不出，只能低头理货假装没听见。
- **可选·清仓-B**：「清仓啦——」喇叭一喊，老客眉头拧起来：上次还原价，这回就甩？他举着手机拍了照，是要发群里让人看你笑话的。

**拖欠房租（delay_rent，+半月租）**
- **可选·拖房租-A**：你给房东发「哥，这月缓几天」，他只回个「？」。你知道这问号比催租还贵——下月房租，他准要找补回来。
- **可选·拖房租-B**：房东半天没回话，中午却"路过"你门口，顺手把价目表拍了张照。那镜头不是闲着，是要给你算总账的前奏。

**拖欠供应商（delay_supplier_payment，+¥3,000）**
- **可选·拖供应商-A**：你跟老供货商说"这月缓一缓"，他"嗯"了声挂了。下周篮里，偏偏少了一样你最离不开的料——他拿捏得准。
- **可选·拖供应商-B**：供应商群里甩来一句「账期能谈，但别让我等太久」。那串省略号吊着，比涨价通知还吓人。

**裁员（layoff，+¥2,500）**
- **可选·裁员-A**：你叫住半年的小伙：「下月不用来了。」他愣了下，把围裙叠整齐放桌上——那围裙，还是你亲手递给他的。
- **可选·裁员-B**：你念完名单，店里静得能听见冰柜嗡一声。留下的人低头擦桌，谁都不敢抬眼，更不敢看你那张脸。

**临时提价（temporary_price_increase）**
- **可选·提价-A**：你悄悄改了招牌套餐价，第一个老客盯菜单愣三秒：「又涨？」你笑着没接，心里算这周能不能活过去。
- **可选·提价-B**：「临时调个价，就几天。」你在收银台贴手写条，落款不敢写真名，只画了个歪笑脸——像在跟老客赔不是。
